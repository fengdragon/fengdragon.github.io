Kindlei 书库 by Calibre2opds

发表于 2013-08-02   |   分类于 Kindle Read

手上搜集着一些用于Kindle上的mobi格式的书籍，
有时和朋友谈论起来，总是要互相分享或发送给对方，
到那时再一个个找又费时又费力，不由得就想利用手头上
能用的资源建起个网站来。

大概的思路是：

  1. 稳定的网络。大小不是大的问题，现在大部分资源都至少1G，或者更多，
    用于电子书，绰绰有余，而且书在于精，在于读，
    1G大小的书也能读上多半年了，哪怕到时候更新工作量也不是很大。

  2. 能够自动生成目录，一般书都会存放在一个目录里，
    手动生成目录以及填入外链实非人力所为。

  3. 方便发布，程序员嘛。。最好用个git push就可以搞定。。

这样子搜了一下，
已有过几个大站，像子午书简这种是使用dropbox+Calibre建立的，
自己的也是采用这种方式，所需要的工具有：

  1. calibre
  2. calibre2opds
  3. openshift

在东京申请澳大利亚商务签

发表于 2013-07-30   |   分类于 Travel Visa

####在东京办理澳大利亚之签证攻略

まあ，天朝子民去哪第一步，签证考虑都要先行，
即使在鬼子国，也毫无例外，往前由于学会要前去墨尔本，
经过一番调查与准备，昨儿终于收到了签证。

签证流程：

  • 准备好所需要的文件资料
  • 邮寄到韩国首尔使馆处
  • 首尔使馆处在接到你的材料后会发一封valid visa application received的信给你
  • 如果需要，可以和使馆处email联系，回复还算及时的
  • 约1~2周会收到审查结果，澳洲不需要往你的护照本上贴visa，收到信件就可以了。
